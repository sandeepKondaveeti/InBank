This is a automated testscript for verifying the below test scenarios.

1. Entering value in to the text box next to search and after submitting, verifying whether it actually searched.
2. Trying to login with empty credentials and verifying the error message is displayed or not.
3. Trying to login with invalid credentials and verifying the error message is displayed or not.


Pre-requisites to have for running the testscript
1. Python installed
2. Place the path of python in Environment variables - C:\Python27\Scripts;
3. Run the command "pip install selenium"
4. Run the script with the following command on command line "python automation_inBank.py"
